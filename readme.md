# 项目简介：
简要介绍项目的目的和功能，可以包含项目的背景和相关的技术。

  该项目是一个基于Redis实现的仿大众点评项目,旨在提供一个平台，供用户分享生活中的美好事物。 项目采用了前后端分离技术，支持用户登录注册、商品查询、优惠卷秒杀下单、用户分享优质店铺、好友关注、用户签到等等一系列功能。
